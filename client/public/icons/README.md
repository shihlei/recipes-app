# Icon Placeholders

Replace `icon-192.png` and `icon-512.png` with real PNG exports before shipping.

Quick generation options:
1. **Squoosh / Sharp** – export `icon.svg` at 192×192 and 512×512 as PNG
2. **Inkscape CLI** – `inkscape icon.svg --export-png=icon-192.png -w 192 -h 192`
3. **Online tools** – realfavicongenerator.net or favicon.io

The SVG source is `icon.svg` in this directory.
